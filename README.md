# spring-boot-rabbitmq-example
Spring boot Mysql RabbitMQ example

##
### Prerequisites
- JDK 1.8 or later
- Maven 3 or later (3.5)
- Mysql
- RabbitMQ

---
## Quick Start

### Clone source
```
git clone https://github.com/jeonguk/spring-boot-rabbitmq-example.git
cd spring-boot-rabbitmq-example
```

### 빌드 & RUN
```
mvn clean install
```
```
mvn spring-boot:run
```
